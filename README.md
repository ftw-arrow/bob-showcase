## 🚀 Quick Start (5 Minutes)

### 1. Create GitHub Repository
```bash
# Go to: https://github.com/new
# Repository name: bob-showcase
# Description: Bob AI Assistant Showcase Site
# Public repository
# Initialize with README
```

### 2. Clone and Set Up
```bash
cd ~/Projects
git clone https://github.com/YOUR_USERNAME/bob-showcase.git
cd bob-showcase

# Create structure
mkdir -p src/{css,js,images,demos}
mkdir -p docs .github/workflows
```

### 3. Deploy to Azure
```bash
# Login
az login

# Create Static Web App (auto-configures GitHub Actions)
az staticwebapp create \
  --name bob-showcase \
  --resource-group rg-bob-showcase \
  --source YOUR_USERNAME/bob-showcase \
  --location eastus2 \
  --branch main \
  --app-location "/src" \
  --login-with-github
```

### 4. Configure DNS
```bash
# Add to GoDaddy DNS:
# CNAME: bob -> <your-app>.azurestaticapps.net
# TXT: _dnsauth.bob -> <validation-token>
```

### 5. Push Initial Site
```bash
# Copy files from implementation guide
# Commit and push
git add .
git commit -m "Initial Bob showcase site"
git push origin main
```
